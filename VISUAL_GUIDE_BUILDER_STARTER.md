### Color Scheme Reference

```
PRIMARY COLORS:
- Cyan (#00D9FF) - Main accent, CTAs
- Purple (#9D4EDD) - Secondary highlight
- Pink (#FF006E or similar) - Brand gradient
- Blue (#3A86FF) - Accent color

BACKGROUNDS:
- Dark glass cards (bg-white/5 to bg-white/10)
- Gradient overlays (from-neon-cyan/5 to purple/5)
- Border accents (border-white/10, border-neon-cyan/50)

TEXT:
- Primary: text-foreground (white)
- Secondary: text-foreground/70 or /60
- Subtle: text-foreground/40
```

### Responsive Breakpoints

```
Mobile (< 768px):
- Single column layouts
- Scrollable tab navigation
- Full-width buttons and inputs
- Stacked form fields

Tablet (768px - 1024px):
- 2-column grids
- Wider cards and sections
- Horizontal form layouts

Desktop (> 1024px):
- Full 3-4 column grids
- Expanded sections
- Optimal spacing and white space
```

## Feature Comparison

| Feature | Business Builder | Brand Starter |
|---------|-----------------|---------------|
| **Purpose** | Create business plans | Create brand strategy |
| **Credit Cost** | 15 credits | 15 credits |
| **Form Steps** | Chat interface | 1 info form |
| **Results** | Interactive chat | Static strategy |
| **Export** | Plan document | Guidelines |
| **AI Integration** | Chat-based advisor | Strategy generation |
| **Route** | `/business-builder` | `/brand-starter` |
| **Dashboard Tab** | Yes | Yes |
| **Templates** | 4 business types | 8 industries |
| **Time to Complete** | 15-30 min | 5-10 min |

## Navigation Flows

### From Dashboard to Features
```
Dashboard
├─ [Business Builder Tab] → /business-builder
│  └─ [Start Building] → Chat Interface
└─ [Brand Starter Tab] → /brand-starter
   └─ [Generate Strategy] → Results Display
```

### From Feature Back to Dashboard
```
Business Builder Page
├─ [Close Button (X)] → Dashboard
├─ [Back to Dashboard] → Dashboard
└─ [Return] after chat end → Dashboard

Brand Starter Page
├─ [Close Button (X)] → Dashboard
├─ [Create Another] → Back to Form
├─ [Go to Dashboard] → Dashboard
└─ [Cancel] → Dashboard
```
