## Mobile Responsive Behavior

### Phone (< 640px)
```
- Single column layout
- Full-width buttons
- Stacked cards
- Scrollable tab navigation
- Reduced padding (24px)
```

### Tablet (640px - 1024px)
```
- 2 column grids
- Medium buttons with padding
- 2-row card layouts
- Tab navigation fits in view
- Medium padding (32px)
```

### Desktop (> 1024px)
```
- 3-4 column grids
- Standard buttons
- Full card layouts
- All tabs visible
- Maximum padding (48px)
```

## Interactive States

### Buttons
```
Default:     bg-gradient (neon-cyan)
Hover:       shadow-lg, shadow-neon-cyan/50
Active:      Slight opacity change
Disabled:    opacity-50, cursor-not-allowed
Loading:     Animated spinner + text change
```

### Input Fields
```
Default:     bg-white/10, border-white/20
Focus:       border-neon-cyan/50
Filled:      text-foreground
Error:       border-red-500
Placeholder: text-foreground/40
```

### Cards
```
Glass:       bg-white/5, border-white/10
Hover:       bg-white/10, border-neon-cyan/50
Selected:    Border highlighted (cyan)
```

---

## Animation & Transitions

### Smooth Transitions
- Duration: 200-300ms
- Easing: ease-in-out (default Tailwind)
- Applied to: Color, border, shadow, opacity

### Loading States
```
Spinner Animation:
- Rotate 360° continuously
- Border: white/20
- Border-top: neon-cyan (transparent)
- Speed: ~1 second per rotation
```

### Hero Section
```
Background Gradients:
- Smooth blur 3xl
- Positioned absolutely
- Opacity: 5% of primary color
- Create depth without distraction
```
